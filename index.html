Introduction to GIT and GITHUB

Git is a distributed version control system.

Git helps keep track of project changes and helps with collaboration.

Common Git Commands

  • git init: initialize a git repository. (makes current project a git project)
  • git add: adds files to staging area. (get them ready to save)
    • (+) plus sign in VSCode's source control window
  • git commit: Saves the git project or takes a snapshot, including everything in staging area.
    • Check Mark Commit in VSCode's source control window
  • git status: Shows the status of the git project
  • git push
  • git pull
  • git clone
  • git branch: creates a new branch
  • git checkout: Switches to different branches or different commits
  • git merge
